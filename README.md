# Huuha Land 评价生成系统 - MVP 版本

## 系统架构

```
用户填写表单
    ↓
AI 生成小红书文案
    ↓
图片 + 文案合成
    ↓
复制到剪贴板
    ↓
跳转到小红书（或显示二维码）
```

## 技术栈

- **前端**: HTML + JavaScript (纯前端，无需后端)
- **AI 生成**: 调用 Claude/GPT API
- **图片合成**: Canvas API
- **部署**: 静态网页 (GitHub Pages / Vercel)

## 核心功能

### 1. 表单页面 (index.html)
- 服务项目选择
- 共鸣点选择
- 发现渠道选择
- 体验描述输入
- 图片上传

### 2. 生成页面 (generate.html)
- 调用 AI API 生成文案
- Canvas 合成图片
- 显示生成结果
- 一键复制文案
- 跳转到小红书按钮

### 3. 预留扩展接口
- share_code 生成（待接入）
- 小红书 SDK 接入（待接入）
